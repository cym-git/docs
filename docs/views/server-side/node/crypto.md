## 散列(哈希)算法

散列算法也叫哈希算法，用来把任意长度的输入换成固定长度的输出，常见的有 md5、sha1 等

- 相同的输入会产生相同的输出
- 不同的输出会产生不同的输出
- 任意的输入长度输出长度都是相同的
- 不能从输出推算输出的值
