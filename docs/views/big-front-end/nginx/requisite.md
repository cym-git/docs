# 前端开发者必备的 nginx 知识

> **nginx**是一个高性能的 HTTP 和反向代理服务器，也是一个通用的 TCP/UDP 代理服务器，最初由俄罗斯人**Igor Sysoev**编写。**nginx**在应用程序中的作用有：
>
> - 解决跨域
> - 请求过滤
> - 配置 gzip
> - 负载均衡
> - 静态资源服务器

## 正向代理与方向代理

> 代理是在服务器和客户端之间架设的一层服务器，代理将接受客户端的请求将它转发给服务器，然后将服务端的响应转发给客户端。不管正向代理还是反向代理，都是实现此功能。
